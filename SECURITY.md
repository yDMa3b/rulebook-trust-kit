# Security

Please report security issues privately by opening a GitHub security advisory or contacting the maintainer listed on the GitHub repository.

Do not use this project to scan or test repositories, systems, or codebases unless you own them or have explicit permission.

This public repository intentionally excludes:

- private product code
- Steam/APK release configuration
- beta feedback or private user data
- API keys, tokens, and environment files
- large commercial media assets
- third-party source mirrors
