## Why Does This Repository Qualify?

```text
Rulebook Trust Kit is an open-source toolkit for mechanically verifiable board/card-game rules: schema-backed rule packages, source-to-rule traceability, implementation/test alignment, edge/semantic examples, and human review packs. It addresses a long-tail preservation problem for regional tabletop games and gives maintainers repeatable quality gates for rule changes.
```

## How Will You Use API Credits?

```text
We will use API credits to run Codex-assisted maintainer workflows: PR review of rule package changes, generation/checking of traceability gaps, review-pack drafting for domain reviewers, release-gate summaries, and regression triage for new game examples. Credits will not be used for private user data or unrelated commercial assets.
```

## Anything Else?

```text
The public repository excludes private product code, Steam/APK assets, beta feedback, environment secrets, and third-party mirrors. The project uses Apache-2.0 for code and CC BY 4.0 for docs/examples.
```
